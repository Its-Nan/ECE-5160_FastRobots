Lab 1: Artemis Setup and Bluetooth

Overview

In this lab, we set up the SparkFun Artemis board and established Bluetooth connectivity. We configured the development environment, uploaded the first sketch, and tested wireless communication between the robot and the computer.

Objectives

  • Set up the Artemis board environment
  • Establish Bluetooth connection
  • Configure wireless communication
  • Test data transmission

Results

Successfully connected the Artemis board to the computer via Bluetooth and transmitted test data. The board was able to receive commands and send sensor readings wirelessly.
